MACD o Convergencia/Divergencia de la media móvil se utiliza para visualizar momentos donde se están marcando ciertos cambios de tendencia ya sea para finalizar una tendencia alcista/bajista o finalizar un pullback y retomar la tendencia principal

![[indicador_macd.png]]

Principalmente lo vamos a utilizar para detectar divergencias con el precio.
Y a que llamamos divergencia, por ejemplo una divergencia seria ver que el precio esta subiendo mientras que la media móvil del MACD esta bajando.
